About this dashboard

Independent replication of the Capital Flows Research STIR Replication Playbook. All math runs in the browser; data is a daily snapshot embedded in the page when it's built.

  • Products tab — SOFR (SR3) / Fed Funds (ZQ) strip toggle, EFFR baseline, terminal contract highlighted.
  • Meetings tab → Strip — day-weighted post-meeting rate path: post = (avg·N − (D−1)·prev) / (N−D+1); FedWatch-style probability table.
  • Meetings tab → Spreads — calendar spread matrix in basis points.
  • Meetings tab → CB LVL — meeting path overlaid with 25 bp policy rails.

Data sources

  • SR3 / ZQ daily settlements: Yahoo Finance (.CBT for ZQ, .CME for SR3).
  • EFFR / SOFR: FRED (EFFR, SOFR).
  • FOMC schedule: hand-maintained in the build script; refresh annually from federalreserve.gov.

Data on this page is "as of last build." To refresh, the maintainer re-runs build_stir.py and pushes.
